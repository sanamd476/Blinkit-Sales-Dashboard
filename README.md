## Project Overview
This project presents an interactive Excel dashboard for analyzing Blinkit grocery sales. The dashboard provides insights into sales performance, outlet characteristics, item categories, and key business metrics using Pivot Tables, Pivot Charts, KPIs, and Slicers.
## Dataset
- Source: Kaggle (Blinkit Grocery Sales Dataset)
- Format: CSV
## Dashboard Features
- Total Sales KPI
- Total Items KPI
- Average MRP
- Average Sales
- Sales by Item Type
- Sales by Outlet Size
- Sales by Fat Content
- Sales by Outlet Type
- Sales Trend by Establishment Year
- Interactive Slicers
## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Business Insights
- Medium-sized outlets generated the highest sales.
- Fruits and Vegetables contributed the highest sales among item categories.
- Low Fat products accounted for the majority of total sales.
- Grocery Stores had the largest share of outlet sales.
